# 快速开始

欢迎使用 Octopus Crawler！本指南将帮助你在 5 分钟内创建第一个数据采集任务。

## 第一步：打开目标网页

在浏览器中打开你想要采集数据的网页。例如，一个电商网站的商品列表页。

## 第二步：启动 Octopus Crawler

点击浏览器工具栏中的 Octopus 图标，打开扩展界面。

## 第三步：创建采集任务

1. 在“采集任务”页面点击"新增"按钮

![新增任务](./img/add-scraper.png)

## 第四步：选择要采集的数据

1. 将鼠标移动到网页上的元素，元素会高亮显示
2. 点击你想要采集的数据，例如商品标题
3. 在弹出的对话框中，选元素 -> 选属性 -> 选数据
4. 重复以上步骤，选择其他要采集的字段（价格、图片等）

![选择字段](./img/select-field.png)

## 第五步：运行采集任务

1. 点击"运行"按钮
2. Octopus 会自动采集当前页面的数据
3. 如果需要采集多页数据，可以启用"自动翻页"功能

![配置自动翻页](./img/pagination.png)

## 第六步：导出数据

1. 采集完成后，在“数据中心”页面对采集的数据点击"导出"按钮
2. 选择导出格式（Excel、CSV、JSON）
3. 保存文件到本地

![导出数据](./img/export-data.png)
