# Installation Guide

## Install from Browser Extension Store

You can install the extension from Chrome Web Store or Firefox Browser Add-ons. After installation, you should restart the browser to make sure the extension has fully loaded. If you don't want to restart browser then use the extension only in tabs that are created after installation.

## Supported Browsers

- Chrome 90+
- Edge 90+
- Firefox 88+
- Other Chromium-based browsers

## Installation Steps

1. Open your browser's extension store
2. Search for "Octopus Crawler"
3. Click "Add to Browser" or "Install" button
4. Confirm installation permissions
5. After installation, you'll see the Octopus icon in your browser toolbar

## Verify Installation

After installation, click the Octopus icon in your browser toolbar. If you can see the extension interface, the installation was successful.

## Common Issues

### Can't find the icon after installation?

Check your browser's extension management page and make sure Octopus Crawler is enabled.

### Extension not working properly?

Try restarting your browser, or uninstall and reinstall the extension.
