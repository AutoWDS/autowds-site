# 安装指南

## 从浏览器扩展商店安装

你可以从 Chrome 应用商店或 Firefox 浏览器附加组件安装扩展。安装后，你应该重启浏览器以确保扩展已完全加载。如果你不想重启浏览器，那么只在安装后创建的标签页中使用扩展。

## 支持的浏览器

- Chrome 90+
- Edge 90+
- Firefox 88+
- 其他基于 Chromium 的浏览器

## 安装步骤

1. 打开浏览器扩展商店
2. 搜索 "Octopus Crawler"
3. 点击"添加到浏览器"或"安装"按钮
4. 确认安装权限
5. 安装完成后，你会在浏览器工具栏看到 Octopus 图标

## 验证安装

安装完成后，点击浏览器工具栏中的 Octopus 图标，如果能看到扩展界面，说明安装成功。

## 常见问题

### 安装后找不到图标？

检查浏览器扩展管理页面，确保 Octopus Crawler 已启用。

### 扩展无法正常工作？

尝试重启浏览器，或者卸载后重新安装。
