# Introduction

## What is AutoWDS?

AutoWDS is a powerful Chrome browser data collection extension plugin. It provides both intelligent collection and advanced collection modes to help users easily extract required data from web pages. The plugin supports visual rule editing, allowing intuitive configuration of data extraction rules through a graphical interface without writing complex code.

## Core Features

### 🤖 Intelligent Collection Mode
Based on AI technology, automatically identifies web page structure and extracts data. Simply right-click on the page and select "Intelligent Collection", the plugin will automatically analyze the page structure and intelligently identify collectible data fields, greatly lowering the barrier to entry.

### 🎨 Visual Rule Editor
Provides an intuitive visual rule editor supporting:
- Click-to-select element selection
- Real-time preview of collection results
- Flowchart display of collection logic
- Drag-and-drop rule configuration

### ⏰ Task Management System
Built-in comprehensive task management features:
- Scheduled tasks (Cron expressions)
- Task execution history
- Error log recording
- Execution result notifications

### 📊 Multi-format Data Export
Collected data can be exported in multiple formats:
- Excel (.xlsx)
- CSV (.csv)
- JSON (.json)
- Google Sheets (cloud sync)

### 🚀 Advanced Collection Capabilities
Supports complex data collection scenarios:
- Batch collection from list pages
- Deep collection from detail pages
- Automatic pagination handling
- Keyword search collection
- Post-login collection
- File and image downloads

No programming background required, complete complex data collection tasks through visual interface. Intelligent collection mode makes data extraction unprecedentedly simple.
