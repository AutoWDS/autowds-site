# 简介

## 什么是八爪鱼爬虫？

八爪鱼爬虫（Octopus Crawler）是一款功能强大的 Chrome 浏览器数据采集扩展插件。它提供智能采集和高级采集两种模式，帮助用户轻松从网页中提取所需数据。插件支持可视化规则编辑，通过图形化界面直观地配置数据提取规则，无需编写复杂代码。

## 核心特性

### 🤖 智能采集模式
基于AI技术，自动识别网页结构并提取数据。只需右键点击页面，选择"智能采集"，插件会自动分析页面结构，智能识别可采集的数据字段，大大降低使用门槛。

### 🎨 可视化规则编辑
提供直观的可视化规则编辑器，支持：
- 点选式元素选择
- 实时预览采集结果
- 流程图展示采集逻辑
- 拖拽式规则配置

### ⏰ 任务管理系统
内置完善的任务管理功能：
- 定时任务（Cron表达式）
- 任务执行历史
- 错误日志记录
- 执行结果通知

### 📊 多格式数据导出
采集的数据可导出为多种格式：
- Excel (.xlsx)
- CSV (.csv)
- JSON (.json)
- Google Sheets（云端同步）

### 🚀 高级采集能力
支持复杂的数据采集场景：
- 列表页批量采集
- 详情页深度采集
- 自动翻页处理
- 关键词搜索采集
- 登录后采集
- 文件和图片下载

## 为什么选择八爪鱼爬虫？

### 🎯 简单易用
无需编程基础，通过可视化界面即可完成复杂的数据采集任务。智能采集模式让数据提取变得前所未有的简单。

### 💪 功能强大
从简单的单页采集到复杂的多级深度采集，从手动执行到定时自动化，满足各种数据采集需求。

### 🔒 安全可靠
所有数据处理都在本地浏览器中完成，不会上传到第三方服务器。支持登录态保持，可以采集需要认证的网站数据。

### 🌐 开放生态
支持与多种第三方服务集成，包括Google Sheets、各类数据库、云存储服务等，方便数据的进一步处理和分析。

### 📈 持续更新
我们持续改进产品功能，修复bug，优化性能，确保用户获得最佳的使用体验。
